How to installation Larvel in local PC

Step 1 - Download the composer https://getcomposer.org/
Step2 - Run cmd
- Please insert the your xampp path example → cd\xampp/htdocs
Step 3 - After finished Step2 you can build your first project file
Step 4 - After Installation complete Open your browser and run URL e.g. http://localhost/your-project-name/public/, you will be see below output.
